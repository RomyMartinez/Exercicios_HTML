Alinhamento, Span e ID

Alinhamento

  • Text-align: É usado para alinhar o texto, left, center, right e justify. text- indent: Afasta o inicio do paragrafo fazendo uma endentação.

Identificadores

ID

  • O id funciona para diferenciar diferentes elementos do html e pode modificar especificamente eles no CSS sem ter que modificar outra tag do mesmo tipo.

  • Para usar ele é feito assim, <alguma tag id="nome">

  • Eu so posso utilizar um id no meu html.

Class

  • Ela também faz a configuração personalizada.

  • Eu utilizo ele assim class="nome". E no CSS identifico ela com um ponto.

  • Posso usar ela quantas vezes eu quiser no meu html.

  • Eu posso colocar mais de uma classe em uma tag, separando ele com um espaço.
