Margin

Margin é a parte que se encontra fora do elemento, a distancia entre um elemento e outro. Para definirmos a margem, utilizamos margin. Se quisermos definir a margem de cada lado colocamos margin: 0 0 0 0;. No sentido do relógio.

Padding

Enquanto padding é a parte de dentro do elemento, a distancia dele até o conteúdo dentro. Para definirmos o padding, utilizamos padding. Se quisermos definir a distancia de cada lado colocamos padding: 0 0 0 0;. No sentido do relógio.

Border-radius

Utilizamos o a propriedade Border-radius para definir o radio da nossa borda. Para definir o radio de cado canto, utilizamos border-radius: 0 0 0 0;. Começa na esquerda de cima até a parte inferior esquerda no sentido do relógio.

Box-shadow

Colocar sombra no elemento. box-shadow: 0px 0px 0px 0px; O primeiro parâmetro é quanto ele anda para a direita, o segundo quanto para baixo, o terceiro quanto ela vai espalhar e o ultimo a transparecia. Também no ultimo você pode escolher a cor em vez da transparecia.
