Exercicio 9

Estilos inline

Estilos inline são aqueles no qual agente coloca dentro das o style="" para modificar o que estiver dentro dela

Estilos internos

Você tem que colocar os estilos dentro de uma tag <style> dentro da tag head

Estilos externos

Pode usar etilos em varias paginas

Nos estilos externos você pode fazer com que varias paginas tenham o mesmo estilo sem te que criar uma tag style no head, criando um arquivo style.css

Para utilizar uma pagina CSS, voce precisar criar uma tag <link> sinalizando o nome da pagina css
