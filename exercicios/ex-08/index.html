Imagens dinâmicas

Utilizamos a tag <picture> para colocar dentro a tag <source> e <img>, sendo a source para a mudar a imagem dependendo do tamanho da tela.

A tag <source> sera a media:type. Também precisamos de colocar os tamanhos em ordem de tamanho.

Reproduzir audio

Para colocar em áudios você precisa da tag <audio>

Dentro do da tag audio você precisa colocar autoplay para o audio reproduzir sozinho e controls para que apareça os ajustes de audio

Também podemos colocar loop para quando acabar a musica ele recomeçar

Outra maneira de colocar o audio

Colocar entre as tags audio colocar a tag source com src e type, você pode colocar vários e sources para se ele nao carregar o primeiro, carregar o outro
